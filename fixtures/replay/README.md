# Replay Fixtures

中文说明：本目录放置确定性回放 fixture。阶段 6 已包含执行只读、模拟成功、部分成交和未知状态路径，用于证明执行计划、模拟报告和模拟账本输入可离线复现。阶段 7 新增对账一致、对账差异和未知状态事故 fixture，用于证明对账和事故记录可离线追溯。阶段 8 新增 `venue_data_smoke/`，用于证明第一个只读场所适配器可离线输出原始行情事件、标准化行情事件、场所健康事件和只读能力样例。

约束：

- 回放输入必须离线、固定、可重复。
- 不访问真实网络，不依赖本机私有状态。
- 不包含密钥、API key、私钥、token 或任何真实凭证。
- 执行阶段 fixture 只包含只读或模拟结果，不包含真实下单、撤单、转账或签名动作。
- 事故 fixture 必须包含 `source_event_refs`，并能按事件 ID 追溯。
