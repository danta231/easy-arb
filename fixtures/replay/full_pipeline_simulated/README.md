# Full Pipeline Simulated Replay Fixture

中文说明：本 fixture 用于阶段 9 `S9-01` 端到端模拟演练。输入只包含离线、
脱敏、公开行情样例和固定组合状态；运行时只能装配既有模块，不连接真实交易
API，不下单、不撤单、不转账、不签名。

输出黄金文件位于 `expected/`。同一输入多次运行必须得到完全一致的候选转换、
风控决策、执行计划、模拟执行报告、模拟账本分录、对账报告和运营日报。

`cases/` 下的异常路径 fixture 覆盖陈旧数据、余额不足和未知状态。中文说明：
这些路径必须在风控拒绝后停止生成执行计划，并输出可追溯事故和运营报告。
