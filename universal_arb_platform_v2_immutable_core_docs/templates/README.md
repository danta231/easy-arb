# 资料模板目录

本目录保存结构化资料模板，不再依赖已清理的 `docs/`（旧开发文档）或 `review/`（旧评审材料）目录。

## 当前文件

- `config.template.yaml`：资料侧配置模板，用于说明配置字段和默认安全姿态。
- `daily_operations_report.template.md`：每日运营报告模板，用于人工查看或文档侧样例。
- `opportunity_record.csv`：机会记录 CSV（逗号分隔值）表头模板，字段名保持稳定英文标识，便于机器读取。
- `opportunity_record.README.md`：机会记录字段说明，提供中文解释。

## 安全边界

- 模板不得包含密钥、接口密钥、私钥、助记词、令牌、会话或 webhook 密钥。
- 默认配置必须保持只读、模拟或预检用途，不能默认打开真实下单、撤单、转账或真实签名。
- 英文技术标识可以保留，但必须提供中文解释。
